# 项目描述

如何从app开发变成后台开发?从选择一个合适的web框架开始吧。
使用[balde](https://github.com/lets-blade/blade)框架开发api，并使用androidstudio 开发了一个简单记事本。实现了简单的crud。


# 使用方式

- 运行服务器  api_project下的Application.java
- 修改app_priject 的 ApiConstants 的 服务器地址
- 编译运行App
